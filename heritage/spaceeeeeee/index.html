This is a sample 3D model for <model-viewer>, which is open source javascript library provided by Google. It is a symbolic model for here, because I planed to make this site when I found this library and sample site. When I saw this awesome library, I wanted to use this library so I looked for where I can use this. then, I came up with idea which is online-museum. When you looking for some heritage, or art, or any object placed in museum, unless you are able to get to a museum, you mostly google that thing or find some book to look into that. But, you can just get shallow information and some pictures that have wierd angle or position so that can't satisfy our purpose. In this case, if we use 3D model to take a look at the object, we can view that in any angle, any position and even we can see what's inside of a object(when if modeler is too sensitive to make a inside of modelüòÄ). It can be useful when we search for the any historical research, Especially students that have many chances to search historical resource like us. So That's because I wanted to make online museum that provide 3D model of heritage. Thanks to visit our website and have a great time!
